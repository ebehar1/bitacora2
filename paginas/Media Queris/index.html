Media Queries

Diferentes resolucionnes de pantallas segun el dispositivo


Telefonos: 320x480px
Tabletas: 768x1024px
Computadoras: 1366x768px o 1920x1080px

Media Querry

¿Que es?


Las media querry son propias de css, nos permiten generar diferentes estilos, acomodos, etc, del contenido de la pagina segun su resolucion de pantalla o el ancho del viewport del navegador. Las media queries se utlizan para lo siguiente:

Para aplicar estilo condicionalmente utilizando reglas en css : @media o @import Poder segmentar medios especificos para style, link, source y otros html con el atributo @media =

BreakPoints


Es un ancho de pantalla específico donde un sitio web o aplicación cambia su diseño para adaptarse a diferentes dispositivos, como teléfonos, tabletas o computadoras de escritorio.

Móviles: entre 320 y 480 píxeles.

Tablets: entre 768 y 1024 píxeles.

Pantallas grandes: más de 1200 píxeles.
